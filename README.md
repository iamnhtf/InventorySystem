# Inventory Management System – Backend API

A simple inventory management backend built with ASP.NET Core.  
This project focuses on core inventory logic such as product management and stock movement tracking.

---

## Features

- Product management (Create, Read)
- Stock In / Stock Out tracking
- Inventory summary per product
- Reorder level warning logic (business-ready)
- RESTful API design
- Swagger API documentation
- Code-first database with EF Core migrations

---

## Tech Stack

- ASP.NET Core (.NET 9)
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI
- RESTful API architecture
